# 하이콰 메뉴 데이터베이스

> 이 파일을 수정하면 웹사이트의 메뉴가 자동으로 업데이트됩니다.
> ID는 자동으로 순차적으로 생성되므로 입력하지 않아도 됩니다!

## 카테고리 ID 매핑

```
1: pho (쌀국수)
2: friedNoodles (볶음면)
3: friedRice (볶음밥)
4: special (스페셜메뉴)
5: side (애피타이져)
6: drink (음료)
7: alcohol (주류)
8: hotfood (안주류)
```

> **필드 입력 방법**
> - **카테고리**: 위 숫자를 입력하세요 (1-8)
> - **가격**: 숫자만 입력하세요 (예: 15000)
> - **추천**, **BEST**: 1을 입력하세요
> - **매운맛**: 0(안매움), 1(매움), 2(더매움), 3(아주매움)
